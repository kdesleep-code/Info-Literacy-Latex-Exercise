## LaTeX とは？
- **TeX**：Donald E. Knuth 氏が開発した組版システム。`\` で始まる命令を使って文書を構築します。  
- **LaTeX**：TeX を拡張したシステム。章・節・表・図などの構造を記述でき、理系文書では標準的に利用されます。  

LaTeX ソースから PDF を作成するには **コンパイル** が必要です。  
一般的な手順は次のとおりです。  

1. `platex` により `.tex` ソースから `.dvi` を生成  
2. `dvipdfmx` により `.dvi` を `.pdf` に変換
